## Abstract

This invention proposes a novel coastline change detection framework based on **multi-scale cascade residual convolutional neural network (CNN)**. The method addresses the limitations of traditional coastline detection approaches, which are often inefficient and labor-intensive. Key components:

1. **Background CNN Module**: Extracts baseline features from satellite imagery
2. **Residual Processing Module**: Enhances feature representation through residual connections
3. **Segmentation CNN Module**: Performs final coastline extraction with fused features

The innovation lies in **deep feature fusion** between the residual processing and segmentation modules, which:
